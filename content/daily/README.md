# Daily Content Packages

Each day's content is stored in `/home/team/shared/daily/YYYY-MM-DD/`

## Structure
```
YYYY-MM-DD/
├── script.md           # Full script for the story
├── short-script.md     # Short-form version (30-60 sec)
├── metadata.md         # Titles, descriptions, hashtags per platform
├── visuals/            # Generated images
│   ├── thumbnail.png
│   ├── story-scene-1.png
│   ├── story-scene-2.png
│   └── instagram-story.png
└── notes.md            # Platform-specific notes
```

## Workflow
1. Content Creator researches and selects story
2. Writes scripts (long + short)
3. Generates visual assets
4. Prepares metadata
5. Packages into daily folder
6. Team lead reviews and approves
7. Content is posted to platforms
